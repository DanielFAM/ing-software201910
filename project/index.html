Proyecto Ingeniería de Software

El proyecto se basa en la creación de una página web que permite realizar diferentes procesos dentro de un hotel, teniendo en cuenta el siguiente enunciado: El dueño de un hotel hotel le pide a usted desarrollar un programa sobre las habitaciones disponibles y reservar habitaciones de su hotel.
El hotel posee tres tipos de habitaciones: simple, doble y matrimonial, y dos tipos de clientes: habituales y esporádicos. Una reserva viene determinada por los datos del cliente, los datos de la habitación, la fecha de entrada y el número de días que estará ocupada la habitación.
